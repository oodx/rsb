üìä EXECUTIVE SUMMARY
================================================================================
The Advanced Progress Dashboard is a sophisticated terminal rendering system for tracking batch/chunk processing with rich, configurable visualization. It provides a multi-row display with state-based coloring, true blinking, and comprehensive progress tracking.

================================================================================
üé® KEY FEATURES
================================================================================
1. Optional Custom Title
   - Can display a custom title above the dashboard
   - Flexible display configuration

2. Advanced Chunk Visualization
   - True ON/OFF blinking for current chunk (~200ms toggle)
   - State-based coloring
     * Failed/Cancelled chunks turn RED
     * Configurable color scheme

3. Comprehensive Status Reporting
   - Success: Single line with GREEN completion details
   - Failure: Two-line RED error and status report
   - Cancellation: Colored multi-line status

4. Terminal Rendering Improvements
   - Clean line clearing for multi-line dashboards
   - Automatic line count handling (4-5 rows)
   - ANSI cursor positioning
